# 📖 Sobre o Projeto

O **EduDataBR** é um projeto de análise de dados educacionais desenvolvido a partir dos microdados oficiais do ENEM, disponibilizados pelo INEP.

O objetivo é construir uma solução completa de dados, aplicando conceitos de Engenharia de Dados, Análise de Dados, Ciência de Dados, Machine Learning e Business Intelligence.

---

# 🎯 Problema de Negócio

Este projeto busca responder perguntas como:

- A renda familiar influencia as notas do ENEM?
- Existe diferença significativa entre escolas públicas e privadas?
- O acesso à internet impacta o desempenho?
- Existem desigualdades regionais?
- É possível prever o desempenho de um candidato?
