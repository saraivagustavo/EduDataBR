# EduDataBR: Análise e Predição de Desempenho no ENEM

## 📚 Sobre o Projeto

O EduDataBR é um projeto de análise de dados educacionais utilizando os microdados oficiais do ENEM disponibilizados pelo INEP.

O objetivo principal é investigar os fatores que impactam o desempenho dos estudantes brasileiros no exame, explorando aspectos sociais, econômicos e educacionais através de técnicas de análise de dados, engenharia de dados e ciência de dados.

O projeto busca unir práticas acadêmicas e profissionais em um pipeline completo de dados, passando por:

- ingestão de dados
- limpeza e tratamento
- análise exploratória
- engenharia de atributos
- modelagem preditiva
- visualização de dados
- dashboard interativo

---

# 🎯 Objetivos

## Objetivo Geral

Analisar os microdados do ENEM para identificar padrões e fatores associados ao desempenho dos candidatos.

## Objetivos Específicos

- Realizar tratamento e limpeza dos dados do ENEM
- Construir análises exploratórias sobre desempenho educacional
- Investigar desigualdades regionais e socioeconômicas
- Criar modelos preditivos de desempenho
- Desenvolver dashboards interativos para visualização dos resultados
